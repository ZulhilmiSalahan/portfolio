# mzulhilmi.github.io

Personal portfolio website for **Muhammad Zulhilmi Md Salahan** — Staff Quality Engineer & AI Testing Specialist.

## 🚀 Live Site
[https://mzulhilmi.github.io](https://mzulhilmi.github.io)

## 🛠 Tech Stack
- **Framework**: Pure HTML5 / CSS3 / Vanilla JS — no build tools required
- **Hosting**: GitHub Pages (auto-deployed via GitHub Actions on push to `master`)
- **Fonts**: Google Fonts (Inter + JetBrains Mono)
- **Icons**: Font Awesome 6

## 📁 Structure
```
mzulhilmi.github.io/
├── index.html          # Main single-page site
├── css/
│   └── portfolio.css   # All styles (dark theme, responsive)
├── js/
│   └── portfolio.js    # Typewriter, scroll animations, parallax
├── .github/
│   └── workflows/
│       └── static.yml  # GitHub Pages auto-deploy
└── README.md
```

## 📦 Deployment
Push to `master` branch — GitHub Actions automatically deploys to GitHub Pages.

```bash
git add .
git commit -m "Update portfolio"
git push origin master
```
